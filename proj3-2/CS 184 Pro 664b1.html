All images below were rendered with the following command: ./pathtracer -t 14 -s 128 -l 1 -m 7 -r 480 360 -f lucy.png ../dae/sky/CBdragon_microfacet_au.dae

As the alpha value increases, the roughness of the surface increases. As it decreases, the glossiness of the macrosurface increases.

Importance sampling

Both images were generated with the following command: ./pathtracer -t 14 -s 64 -l 1 -m 7 -r 480 360 -f lucy.png ../dae/sky/CBbunny_microfacet_cu.dae

In the default cosine hemisphere sampling implementation, there is clearly much more noise in the final rendered image due to undersampling around the bunny. To solve this, we can either generate more samples, since the default implementation takes longer to converge to a noise-free result, or we use importance sampling to improve the estimate quality by sampling according to the shape of the Beckmann NDF. As shown in the images above, it is clear that the latter approach generates a much less noisy result.
